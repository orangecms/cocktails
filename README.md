# Cocktail Recipes

A small collection of very simple :cocktail: recipes.
Usually for tall, 400ml glasses. Improvise as necessary.

## Programmer's Poison

This will make a nice, dark green, poisonous looking mixture.
And it's sweeter than honey. Stir well.

- crushed ice
- 300 ml cola
- 4cl blue curacao

## Debugger's Delusion

Delightful and exciting it starts, just like a true debug session.

- crushed ice
- 300ml mate iced tea
- 4cl absinthe

## Coder's Cocktail

Just in case you need to get really serious.

- ice cubes
- 2cl each vodka, gin, rum, tequila, herbal liqueur
- 250ml energy drink
